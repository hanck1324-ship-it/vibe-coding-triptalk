## 이미지 다운로드 방법

1. 피그마에서 해당 노드 ID를 찾아 선택
2. 오른쪽 패널에서 "Export" 클릭
3. PNG 형식, 2x scale로 다운로드
4. `src/assets/` 폴더에 위의 파일명으로 저장

또는 피그마 MCP 도구를 사용하여 자동으로 다운로드할 수 있습니다.
