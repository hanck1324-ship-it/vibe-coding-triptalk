아래의 조건을 모두 적용하여, 아래의 요구사항을 모두 구현할 것.
구현 결과를 체크리스트로 반환할 것.

==============================================

조건-커서룰) 아래의 커서룰을 적용하여 작업하고, 이 작업이 끝나면 해당 rules 적용 결과를 체크리스트로 반환할 것.
            - @01-common.mdc
            - @02-wireframe.mdc

==============================================

조건-파일경로) 구현될 페이지 파일경로: src/app/boards/[boardId]/page.tsx
조건-파일경로) 참고할 컴포넌트 경로:
            - src/components/boards-detail/detail/index.tsx (게시글 상세)
            - src/components/boards-detail/comment-list/index.tsx (댓글 목록)
            - src/components/boards-detail/comment-write/index.tsx (댓글 작성)

==============================================

핵심요구사항) 게시글 상세 페이지의 전체 레이아웃 구조를 step-by-step 으로 구현할 것.

        1) 페이지 구조
            - 게시글 상세 컴포넌트 (detail)
            - 댓글 목록 컴포넌트 (comment-list)
            - 댓글 작성 컴포넌트 (comment-write)
            - 3개 컴포넌트를 수직으로 배치

        2) 스타일링 요구사항
            - 간단한 컨테이너로 감싸기
            - flexbox 레이아웃 사용
            - 각 컴포넌트 사이 적절한 여백
