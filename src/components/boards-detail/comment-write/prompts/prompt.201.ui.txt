핵심요구사항) 댓글 작성 UI를 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

        1) UI 구조 요구사항
            - 1. 헤더 영역
                - 댓글 아이콘 및 "댓글" 텍스트 표시
                - 아이콘: @/assets/chat.png
            
            - 2. 별점 영역
                - Ant Design Rate 컴포넌트 사용
                - 작성자/비밀번호 입력창 위에 배치
            
            - 3. 입력 영역
                - 작성자 입력 (input, type: text, placeholder: "작성자")
                - 비밀번호 입력 (input, type: password, placeholder: "비밀번호")
                - 두 입력창은 가로로 배치
            
            - 4. 내용 영역
                - 내용 입력 (textarea, placeholder: "내용을 입력해주세요.", maxLength: 100)
                - 글자 수 표시 (현재 글자 수/100)
                - 등록하기 버튼
            
            - 5. 버튼 영역
                - 등록하기 버튼 (우측 하단)

        2) 데이터 바인딩 요구사항
            - hook.ts에서 만든 useCommentWrite 훅 사용
            - writer, password, contents, rating 상태 바인딩
            - onChange 핸들러 연결

        3) 기능 요구사항
            - 작성자, 비밀번호, 내용 입력 처리
            - 별점 선택 처리
            - 등록하기 버튼 클릭 시 제출
            - 성공 시 입력창 초기화

        4) 스타일링 요구사항
            - CSS Modules 사용 (styles.module.css)
            - flexbox 레이아웃 활용
            - 입력창 스타일 일관성 유지
