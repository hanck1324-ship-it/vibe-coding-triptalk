조건-파일경로) 구현될 CSS 파일경로: src/components/boards-detail/comment-list/styles.module.css

==============================================

핵심요구사항) 댓글 목록의 와이어프레임 구조를 구현할 것
            (구조와 배치만 정의, 색상/크기/폰트 등 스타일은 prompt.201에서 처리)

        1) HTML 구조 및 레이아웃 배치

            A. 댓글 목록 컨테이너
               - 댓글 아이템들을 감싸는 컨테이너
               - 레이아웃: display: flex
               - 방향: flex-direction: column
               - 아이템 간격: gap 적용

            B. 댓글 아이템 (각 댓글별로 반복)

               1) 상단 영역
                  - 레이아웃: display: flex
                  - 방향: flex-direction: row
                  - 정렬: align-items: center
                  - 간격: gap 적용

                  구성 요소:
                  - 프로필 이미지
                  - 작성자 정보 영역 (flex: 1)
                    * 작성자 이름
                    * 별점 (Ant Design Rate 컴포넌트)
                  - 액션 버튼 영역
                    * 수정 아이콘 버튼
                    * 삭제 아이콘 버튼

               2) 댓글 내용 영역
                  - 레이아웃: display: block
                  - 댓글 텍스트 표시

               3) 하단 정보 영역
                  - 작성일시 표시

               4) 구분선
                  - 마지막 댓글 제외, 각 댓글 하단에 구분선 배치

        2) 기술 요구사항
            - CSS Modules 사용 (styles.module.css)
            - flexbox 레이아웃 사용
            - 구조와 배치만 구현 (레이아웃 방식, flex 설정)
            - 스타일링 요소는 제외 (색상, 크기, 폰트 등은 prompt.201 참조)

        3) 컴포넌트 타입 결정
            - 프로필 이미지: Next.js Image 컴포넌트
            - 별점: Ant Design Rate 컴포넌트 (disabled 모드)
            - 아이콘 버튼: Next.js Image 컴포넌트 (클릭 가능)
            - 구분선: div 요소

        4) 주의사항
            ⚠️ wireframe 단계에서 포함하지 말 것:
            - 색상 값 (hex, rgb 등)
            - 크기 값 (px, rem 등의 구체적 수치)
            - padding, margin 구체적 수치
            - font-size, font-weight
            - border-radius 값
            - 상태별 스타일 (hover, focus, disabled 스타일)

            ✅ wireframe 단계에서 반드시 포함할 것:
            - display 속성 (flex, block)
            - flex 레이아웃 설정 (flex-direction, align-items)
            - 요소 배치 방향 (row, column)
            - 정렬 방식 (align-items, justify-content)
            - 컴포넌트 타입 결정
