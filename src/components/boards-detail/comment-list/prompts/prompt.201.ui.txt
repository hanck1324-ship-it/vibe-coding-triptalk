핵심요구사항) 댓글 목록 UI를 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

        1) UI 구조 요구사항
            - 1. 댓글 아이템 영역
                - 프로필 이미지 표시 (@/assets/profile_image.png)
                - 작성자 이름 표시
                - 별점 표시 (Rate 컴포넌트, disabled 모드)
                - 수정 아이콘 (@/assets/pencil.png)
                - 삭제 아이콘 (@/assets/trashbin.png)
            
            - 2. 댓글 내용 영역
                - 댓글 내용 표시
                - 작성일시 표시 (YYYY.MM.DD 형식)
            
            - 3. 구분선
                - 댓글 아이템 간 구분선 표시 (마지막 댓글 제외)

        2) 데이터 바인딩 요구사항
            - hook.ts에서 만든 useCommentList 훅 사용
            - fetchBoardComments 데이터를 map으로 순회하며 렌더링
            - 각 댓글의 writer, rating, contents, createdAt 표시

        3) 기능 요구사항
            - 댓글 목록 조회 및 표시
            - 수정 아이콘 클릭 시 수정 모달/폼 표시 (향후 구현)
            - 삭제 아이콘 클릭 시 삭제 확인 및 실행 (향후 구현)

        4) 이미지 및 아이콘
            - 프로필 이미지: @/assets/profile_image.png
            - 수정 아이콘: @/assets/pencil.png
            - 삭제 아이콘: @/assets/trashbin.png
            - Next.js Image 컴포넌트 사용

        5) 스타일링 요구사항
            - CSS Modules 사용 (styles.module.css)
            - flexbox 레이아웃 활용
            - 댓글 아이템 간 간격 및 구분선 처리
