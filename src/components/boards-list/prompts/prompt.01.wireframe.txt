            - @01-common.mdc
            - @02-wireframe.mdc

==============================================

조건-파일경로) 구현될 TSX 파일경로: src/components/boards-list/index.tsx
조건-파일경로) 구현될 CSS 파일경로: src/components/boards-list/styles.module.css

==============================================

핵심요구사항) HTML과 flexbox를 활용한 와이어프레임 구조만 만들어낼 것.

핵심요구사항) 게시글 목록의 기본 레이아웃 구조를 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

        1) 레이아웃 구조
            - 1. wrapper 영역
                - 전체 게시글 목록을 감싸는 컨테이너
            
            - 2. 게시글 아이템 영역
                - 각 게시글을 표시하는 영역
                - 작성자, 제목, 작성일시 표시 공간
            
            - 3. 페이지네이션 영역
                - 이전/다음 버튼 영역

        2) 스타일링 요구사항
            - CSS Modules 사용 (styles.module.css)
            - flexbox 레이아웃만 사용 (position: absolute 금지)
            - 최소한의 스타일만 적용 (구조 위주)
