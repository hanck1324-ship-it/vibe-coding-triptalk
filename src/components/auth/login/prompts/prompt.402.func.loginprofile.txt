핵심요구사항) 로그인 후 프로필 표시 및 드롭다운 메뉴 기능을 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

        1) 라이브러리조건
            - API: @apollo/client의 useQuery 훅을 사용할 것.
            - 타입: @/commons/graphql/graphql에서 codegen으로 생성된 타입을 사용할 것.
            - 상태 관리: React의 useState, useEffect 훅을 사용할 것.
            - 라우팅: next/navigation의 useRouter 훅을 사용할 것.
            - 이미지: next/image를 사용할 것.
            - 저장소: localStorage 사용

        2) 기능 요구사항 1 - 로그인 후 프로필 표시
            - 1. (GQL) fetchUserLoggedIn 쿼리를 사용할 것.
                - 파일 위치: src/components/mypage/queries.ts 참조 또는 navigation hook에서 직접 사용
                - Query: fetchUserLoggedIn
                - 응답데이터:
                    - _id
                    - name (사용자 이름)
                    - email
                    - picture (프로필 이미지 URL)
                    - userPoint { amount } (포인트)
                    - createdAt

            - 2. 로그인 상태 확인 및 사용자 정보 표시
                - localStorage에서 accessToken 확인
                - accessToken이 있으면 fetchUserLoggedIn 쿼리 실행
                - 프로필 섹션 구성:
                    - 프로필 이미지 (32x32px, 원형)
                    - 사용자 이름 표시
                - 로그인 전: "로그인" 버튼 표시
                - 로그인 후: 프로필 정보 표시

        3) 기능 요구사항 2 - 프로필 드롭다운 메뉴
            - 1. 드롭다운 상태 관리
                - useState로 드롭다운 열림/닫힘 상태 관리
                - 프로필 클릭 시 토글 함수 생성
                - 드롭다운 외부 클릭 시 닫기 (useEffect와 document click event 사용)
                - ESC 키 누를 시 닫기

            - 2. 드롭다운 메뉴 구성 (4개 항목)
                - 항목 1: 프로필 이름
                    - 사용자 이름 텍스트 표시
                    - 클릭 불가 (정보 표시용)

                - 항목 2: 포인트
                    - "보유 포인트: {amount.toLocaleString()} P" 형식으로 표시
                    - 클릭 불가 (정보 표시용)

                - 항목 3: 포인트 충전
                    - "포인트 충전" 텍스트 표시
                    - 클릭 시 "/myPage?tab=point" 페이지로 이동
                    - 마우스 hover 시 배경색 변경

                - 항목 4: 로그아웃
                    - "로그아웃" 텍스트 표시
                    - 클릭 시 로그아웃 함수 실행 (prompt.401.func.logout.txt 기능 통합)
                    - localStorage에서 accessToken 제거
                    - 로그인 페이지로 리다이렉트
                    - 마우스 hover 시 배경색 변경

            - 3. 드롭다운 스타일링
                - 위치: 프로필 섹션 바로 아래에 표시 (position: absolute)
                - 배경: 흰색 (#ffffff)
                - 테두리: 1px solid #e8e8e8
                - 그림자: box-shadow 적용
                - 둥근 모서리: border-radius 6px
                - 각 항목 간 구분선: border-bottom 또는 적절한 간격
                - 메뉴 항목 padding: 12px 16px
                - hover 시 배경색: #f5f5f5
                - z-index: 충분히 높게 설정 (예: 1000)

        4) 기능 요구사항 3 - Figma 디자인 적용
            - Figma API 또는 MCP를 통해 디자인 스펙 가져오기
            - 로그인 후 프로필 표시 영역:
                - Figma에서 지정한 프로필 이미지 크기, 간격, 폰트 스타일 적용
                - 색상 값 정확히 반영

            - 프로필 드롭다운 메뉴:
                - Figma에서 지정한 메뉴 항목 높이, padding, 간격 적용
                - hover 색상, 텍스트 스타일 반영
                - 드롭다운 위치 및 크기 Figma 스펙과 일치

        5) 에러 처리 요구사항
            - 사용자 정보 조회 실패 시:
                - 에러 로깅

                - 로그인 버튼 표시

            - 네트워크 에러 처리:
                - 적절한 에러 메시지 표시 또는 로깅
