구현 결과를 체크리스트로 반환할 것.

            - @01-common.mdc
            - @02-wireframe.mdc

==============================================

조건-파일경로) 구현될 TSX 파일경로: src/components/auth/login-index.tsx
조건-파일경로) 구현될 CSS 파일경로: src/components/auth/login-stylesmodules.css

==============================================

핵심요구사항) HTML과 flexbox를 활용한 와이어프레임 구조만 만들어낼 것.

핵심요구사항) 로그인 폼의 기본 레이아웃 구조를 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

        1) 레이아웃 구조
            - 1. wrapper 영역
                - 전체 로그인 폼을 감싸는 컨테이너
            
            - 2. 로고 영역 (선택사항)
                - 로고 이미지 또는 텍스트 표시 공간
            
            - 3. 입력 영역
                - 이메일 입력 필드
                - 비밀번호 입력 필드
            
            - 4. 버튼 영역
                - 로그인 버튼
                - 회원가입 링크 또는 버튼 (선택사항)

        2) 스타일링 요구사항
            - CSS Modules 사용 또는 전역 CSS 사용
            - flexbox 레이아웃만 사용 (position: absolute 금지)
            - 최소한의 스타일만 적용 (구조 위주)
