핵심요구사항) HTML과 flexbox를 활용한 와이어프레임 구조만 만들어낼 것.

핵심요구사항) 마이페이지 네비게이션의 기본 레이아웃 구조를 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

        1) HTML 구조
            - 1. 마이페이지 부모 영역 (myPageContainer)
                - 전체 마이페이지 네비게이션을 감싸는 최상위 컨테이너
                - display: flex
                - flex-direction: column
            
            - 2. 내 정보 Wrapper (infoWrapper)
                - 내 정보 영역을 감싸는 래퍼
                - display: flex
                - flex-direction: column
            
            - 3. 내 정보 영역 (infoSection)
                - display: flex
                - flex-direction: column
                - 다음의 하위 섹션들을 포함:
                    a. 프로필 섹션 (profileSection)
                        - display: flex
                        - flex-direction: row
                        - align-items: center
                        - 프로필 이미지
                        - 프로필 이름
                    
                    b. 구분선 (divider)
                    
                    c. 포인트 섹션 (pointSection)
                        - display: flex
                        - flex-direction: row
                        - align-items: center
                        - 포인트 아이콘
                        - 포인트 값
                    
                    d. 구분선 (divider)
                    
                    e. 메뉴 섹션 (menuSection)
                        - display: flex
                        - flex-direction: column
                        - 메뉴 항목들 (button 요소)
                            - 거래내역&북마크
                            - 포인트 사용 내역
                            - 비밀번호 변경
                        - 각 메뉴 항목:
                            - display: flex
                            - flex-direction: row
                            - align-items: center
                            - justify-content: space-between
                            - 메뉴 텍스트
                            - 오른쪽 화살표 아이콘

        2) 레이아웃 방식
            - 모든 영역은 flexbox 레이아웃 사용
            - position: absolute 금지
            - 내 정보 영역은 세로 방향으로 배치 (flex-direction: column)
            - 프로필 섹션과 포인트 섹션은 가로 방향으로 배치 (flex-direction: row)
            - 메뉴 섹션은 세로 방향으로 배치 (flex-direction: column)
            - 메뉴 항목은 가로 방향으로 배치 (flex-direction: row)

        3) 반응형 구조 변경
            - 데스크톱: 내 정보 영역 고정 너비 유지
            - 모바일 (768px 이하): 내 정보 영역 전체 너비로 변경

        4) 컴포넌트 타입 결정
            - 프로필 이미지: Image 컴포넌트 (Next.js Image)
            - 포인트 아이콘: Image 컴포넌트 (Next.js Image)
            - 메뉴 항목: button 요소
            - 화살표 아이콘: Image 컴포넌트 (Next.js Image)
            - 구분선: div 요소

        5) 스타일링 요구사항 (최소한만)
            - CSS Modules 사용 (styles.module.css)
            - flexbox 레이아웃만 사용 (position: absolute 금지)
            - 최소한의 스타일만 적용 (구조 위주)
            - 색상, 크기, 간격 등 구체적 수치는 포함하지 말 것
