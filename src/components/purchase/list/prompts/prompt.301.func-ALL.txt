        1) 라이브러리
            - React: useState, useEffect, useMemo, useCallback
            - Apollo: useQuery, useMutation
            - Next: useRouter
            - Ant Design: Button, Input, DatePicker, Modal
            - Icons: @ant-design/icons (HeartOutlined, HeartFilled, SearchOutlined, DeleteOutlined, LeftOutlined, RightOutlined)

        2) GraphQL 쿼리/뮤테이션 (queries.ts)
            - FETCH_TRAVELPRODUCTS: 상품 목록 조회 (search, page, isSoldout)
            - FETCH_USER_LOGGED_IN: 로그인 사용자 정보 (@/components/mypage/queries에서 import)
            - DELETE_TRAVELPRODUCT: 상품 삭제

        3) 핵심 기능 (hook.ts)

            [A] 데이터 조회
                - fetchTravelproducts 쿼리로 상품 목록 조회
                - fetchUserLoggedIn 쿼리로 사용자 정보 조회
                - useMemo로 탭별 필터링 (soldAt 유무)

            [B] 탭 & 검색 & 필터
                - activeTab: "available" | "closed"
                - searchKeyword: string
                - selectedDate: Dayjs | null
                - selectedCategories: string[]
                - handleSearch: refetch 호출

            [C] 북마크 (localStorage)
                - bookmarkedIds: string[]
                - localStorage에서 'bookmarkedProducts' 저장/로드
                - handleToggleBookmark(productId)

            [D] 삭제 (본인 상품만)
                - loggedInUserId와 seller._id 비교
                - 삭제 모달 표시
                - deleteTravelproduct 뮤테이션
                - refetch로 목록 갱신

            [E] 최근 본 상품 (localStorage)
                - recentProducts: string[] (최대 3개)
                - localStorage에서 'recentProducts' 저장/로드
                - addRecentProduct(productId): 중복 제거 후 맨 앞 추가

            [F] 슬라이더
                - recommendCardIndex, bannerIndex: number
                - handleNextRecommend, handlePrevRecommend
                - handleNextBanner, handlePrevBanner
                - useEffect로 배너 자동 슬라이드 (5초마다)

            [G] 페이지네이션
                - currentPage: number (초기값 1)
                - itemsPerPage: 8
                - totalPages: Math.ceil(상품수 / itemsPerPage)
                - paginatedProducts: useMemo로 계산
                - handleNextPage, handlePrevPage

        4) UI 구현 (index.tsx)

            [A] 추천 카드 영역
                - 북마크 버튼 (우측 상단)
