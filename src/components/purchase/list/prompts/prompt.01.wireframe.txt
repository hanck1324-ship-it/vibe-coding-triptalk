핵심요구사항) HTML과 flexbox를 활용한 와이어프레임 구조만 만들어낼 것.

핵심요구사항) 숙박권 구매 페이지의 기본 레이아웃 구조를 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

            
        1) 레이아웃 구조 
            - 1. wrapper 영역
                - 전체 숙박권 구매 페이지를 감싸는 컨테이너
            
            - 2. 네비게이션 영역
                - 기존의 commons 폴더의 것을 확인하여 구현
            
            - 3. 헤더 영역 
                - 페이지 제목 영역
                - 필터/검색 영역
            
            - 4. 상품 추천 영역
                - 2장의 숙박권 카드 배열 (좌우 배치)
            
            - 5. 광고 배너 영역
                - 배너 이미지 표시 공간
            
            - 6. 숙박권 목록 영역
                - 숙박권 카드 그리드 (4열)
                - 각 카드: 이미지 + 제목 + 설명 + 가격
            
            - 7. 페이지네이션 영역
                - 이전/다음 버튼 영역
            
            - 8. 최근 본 상품 영역
                - 사이드 영역에 표시될 공간

        2) 스타일링 요구사항
            - CSS Modules 사용 (styles.module.css)
            - flexbox 레이아웃만 사용 (position: absolute 금지)
            - 최소한의 스타일만 적용 (구조 위주)
            - 실제 이미지/데이터 대신 플레이스홀더 텍스트 사용
