핵심요구사항) 숙박권 판매 등록 폼 UI를 step-by-step 으로 구현할 것.

        1) UI 구조 요구사항
            - 이미지 업로드 영역 (node-id=285-31865)
              * 이미지 크기: 피그마 디자인 참조
              * 파일 타입 제한: JPG, JPEG, PNG, WebP
              * 파일 크기 제한: 개당 제한 용량 이하
              * 최대 업로드 개수: 피그마 디자인 참조
              * 썸네일 크기: 피그마 디자인 참조
              * 삭제 버튼: 각 이미지 오른쪽 상단 X 동그라미 (피그마 디자인 참조)
              * 갤러리 형태로 옆으로 나열
              * 추가 버튼 배경: #f2f2f2 (연한 회색) ⭐
            
            - 기본 입력 필드들 (전체 너비)
              * 상품명, 한줄요약, 상품설명, 판매 가격
              * 각 필드 사이 구분선 (width: 100%, 피그마 디자인 참조, #e4e4e4)
            
            - 주소/위도/경도 + 지도 (2-Column 레이아웃) node-id=285-31848
              왼쪽 컬럼:
                * 주소 (우편번호 + 검색 버튼, 피그마 디자인 참조)
                * 상세주소 (라벨 없음)
                * 위도(LAT)
                * 경도(LNG)
              
              
              오른쪽 컬럼:
                * 상세위치 라벨
                * 지도 영역 (피그마 디자인 참조)
                * border-radius: 피그마 디자인 참조
            
            - 버튼 영역
              * 취소 버튼: 흰색 배경, 검은색 테두리
              * 등록 버튼: 회색(#c7c7c7) → 파란색(#2974e5)

        2) 스타일링 요구사항 (상세)
            - CSS Modules 사용 (styles.module.css)
            - flexbox 레이아웃
            
            - 연한 회색 통일 (#f2f2f2) ⭐매우 중요⭐
              * 위도(LAT) disabled 배경: #f2f2f2
              * 경도(LNG) disabled 배경: #f2f2f2
              * 상세위치 지도 배경 (주소 없을 때): #f2f2f2
              * 사진 업로드 추가 버튼 배경: #f2f2f2
              * disabled 텍스트 색상: #999999 (모두 동일)
            
            - 구분선 스타일
              * width: 100% (컨테이너에 맞춤, 절대 넘어가지 않음)
              * height: 피그마 디자인 참조
              * background: #e4e4e4
              * align-self: stretch
            
            - 버튼 상태별 색상
              * 등록 버튼 disabled: 배경 #c7c7c7, 텍스트 #e4e4e4, cursor: not-allowed
              * 등록 버튼 enabled: 배경 #2974e5, 텍스트 흰색, hover 시 #1e5bb8
              * 취소 버튼: 배경 흰색, 테두리 검은색, hover 시 #f9f9f9
            
            - 2-Column 레이아웃 (주소/위도/경도 + 지도)
              * 전체 컨테이너: display: flex, gap: 피그마 디자인 참조
              * 왼쪽 컬럼: width: 피그마 디자인 참조, flex-direction: column, gap: 피그마 디자인 참조
              * 오른쪽 컬럼: flex: 1, max-width: 피그마 디자인 참조
              * 지도 컨테이너: width: 피그마 디자인 참조, height: 100%, flex: 1
              * ⚠️ 왼쪽 컬럼 내부에는 구분선 없음! gap만 사용
            
            - 입력 필드 스타일
              * 기본: 배경 #ffffff, 테두리 #d4d3d3
              * focus: 테두리 #2974e5
              * disabled: 배경 #f2f2f2, 텍스트 #999999, 테두리 없음
              * border-radius: 피그마 디자인 참조
              * padding: 피그마 디자인 참조
              * font-size: 피그마 디자인 참조
            
            - 지도 영역 스타일
              * border-radius: 피그마 디자인 참조
              * 우편번호 없을 때 배경: #f2f2f2
              * placeholder 텍스트: #999999
            
            - 유효성 검증 에러 표시
              * 색상: #f66a6a (빨간색)
              * 해당 필드 아래 표시

        3) 라이브러리 사용 (필요시)
            - Ant Design 컴포넌트 사용 가능
              * Modal: 주소 검색, 알림 등
              * Upload: 이미지 업로드 (선택사항)
              * DatePicker: 날짜 선택 (향후 필요 시)
            - boards-write 참고하여 일관성 있게 적용
