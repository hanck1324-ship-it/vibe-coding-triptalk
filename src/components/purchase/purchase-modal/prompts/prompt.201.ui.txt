핵심요구사항) 숙박권 구매 모달 UI를 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

        1) UI 구조 요구사항
            
            [A] 모달 기본 구조
                - Ant Design Modal 컴포넌트 사용
                - Props: isOpen, onClose, accommodationId, accommodationData
                - accommodationData에는 제목, 가격, 이미지 포함
            
            [B] 숙박권 요약 정보
                - 숙박권 이미지 (썸네일)
                - 숙박권 제목
                - 1박 가격 표시
                - 총 결제 금액 강조
            
            [C] 구매자 정보 입력 폼
                - 이름 입력 필드 (필수)
                - 연락처 입력 필드 (필수)
                - 이메일 입력 필드 (선택)
                - 체크인/체크아웃 날짜 선택
            
            [D] 결제 정보 입력 폼
                - 결제 방법 선택 (라디오 버튼)
                  - 카드 결제
                  - 계좌 이체
                  - 간편 결제
                - 약관 동의 체크박스
            
            [E] 버튼 영역
                - 취소 버튼: onClick → onClose() 호출
                - 구매하기 버튼: 
                  - 필수 입력값 검증
                  - 성공 시 → 구매 완료 페이지로 라우팅
                  - 라우팅 경로: /purchase/complete/[orderId]
                  - useRouter의 push 메서드 사용

        2) 데이터 바인딩 요구사항
            - accommodationData props로 받은 데이터 표시
            - 구매자 정보는 상태로 관리 (부모에서 전달받거나 내부 관리)
            - 유효성 검증: 필수 필드 공백 체크

        3) 스타일링 요구사항
            - CSS Modules 사용
            - 모달 너비: 600px
            - 내부 패딩 및 여백 일관성
            - 필수 필드 표시 (빨간 별표)
            - 에러 메시지 표시 영역
            - 버튼 hover 효과
            - global.css 토큰 활용

        4) 인터랙션 요구사항
            - 모달 외부 클릭 시 닫기
            - ESC 키로 닫기
            - 구매하기 버튼 클릭 시:
              1. 필수 입력값 검증
              2. 검증 성공 → API 호출 (TODO)
              3. 성공 시 → /purchase/complete/[orderId] 이동
              4. 실패 시 → 에러 메시지 표시

        5) Props 인터페이스
            - isOpen: boolean (모달 표시 여부)
            - onClose: () => void (닫기 콜백)
            - accommodationId: string (숙박권 ID)
            - accommodationData: {
                title: string
                price: number
                image: string
              }

        6) 테스트 체크리스트
            - [ ] 모달이 정상적으로 열리고 닫히는가?
            - [ ] 숙박권 정보가 올바르게 표시되는가?
            - [ ] 필수 입력 필드 검증이 작동하는가?
            - [ ] 구매하기 버튼 클릭 시 라우팅이 되는가?
            - [ ] 취소 버튼으로 모달이 닫히는가?
