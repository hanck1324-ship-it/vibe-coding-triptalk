핵심요구사항) 숙박권 리뷰 목록 컴포넌트 UI를 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

        1) 핵심 UI 구조

            A. 리뷰 통계 영역
               - 평균 평점 (큰 글씨로 강조)
               - 전체 리뷰 수
               - 별점별 분포 막대 그래프 (선택사항)

            B. 정렬 옵션 영역
               - 최신순 / 평점 높은순 / 평점 낮은순
               - 드롭다운 또는 버튼 그룹

            C. 리뷰 아이템 (반복)
               - 작성자 정보:
                 * 프로필 이미지 (40x40px, 원형)
                 * 작성자 이름
                 * 작성일 (YYYY.MM.DD 형식)

               - 평점:
                 * 별점 표시 (antd Rate, disabled)

               - 리뷰 내용:
                 * 본문 텍스트
                 * 최대 3줄 표시, 더보기 버튼

               - 리뷰 이미지 (있으면):
                 * 썸네일 갤러리 (3-4개)
                 * 클릭 시 확대 보기

               - 액션 버튼:
                 * 도움이 됐어요 버튼 (좋아요 카운트 포함)
                 * 신고하기 버튼 (작은 텍스트 링크)

            D. 페이지네이션
               - antd Pagination 컴포넌트
               - 페이지당 10개 표시

            E. 빈 상태 (리뷰 없음)
               - "아직 작성된 리뷰가 없습니다" 메시지
               - 첫 리뷰 작성 유도 버튼

        2) 스타일링 요구사항
            - CSS Modules 사용 (styles.module.css)
            - 리뷰 아이템: 카드 스타일 (border, padding: 20px)
            - 아이템 간 간격: 16px
            - 프로필 이미지: 40x40px, border-radius: 50%
            - 평점: 읽기 전용, 노란색 별
            - 도움됐어요 버튼: hover 시 색상 변경
            - 반응형: 모바일에서 padding, 이미지 크기 조정

        3) Hook 연동 (hook.ts)
            - useCommentList hook 구현
            - Props:
              * accommodationId: number
            - 상태 관리:
              * reviews: Review[]
              * totalCount: number
              * averageRating: number
              * currentPage: number
              * sortBy: 'latest' | 'highRating' | 'lowRating'
              * isLoading: boolean
            - 함수:
              * fetchReviews: 리뷰 목록 조회
              * handlePageChange: 페이지 변경
              * handleSortChange: 정렬 변경
              * handleLike: 도움됐어요 클릭

        4) Mock 데이터 활용
            - mockData.ts의 Review 인터페이스 사용
            - 초기 개발 시 mock 데이터로 UI 테스트

        5) 인터랙션
            - 더보기 버튼 클릭 시 전체 내용 펼치기
            - 이미지 클릭 시 확대 모달 (선택사항)
            - 도움됐어요 버튼 클릭 시 카운트 증가/감소
