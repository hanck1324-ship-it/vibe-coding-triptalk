조건-파일경로) 구현될 CSS 파일경로: src/commons/layout/banner/styles.module.css
조건-파일경로) 아이콘 파일경로: src/assets/close.png (닫기 아이콘)

조건-공통목록) 사용할 색상 목록(global.css에 명시된 변수 토큰 활용)
            - 배경색: 비슷한 primary 또는 accent 색상 토큰 선택하여 사용
            - 텍스트색: white 또는 텍스트 관련 토큰 사용
            - 호버색: 버튼 호버 시 투명도 조정

조건-공통목록) 사용할 타이포그래피 목록(global.css에 명시된 변수 토큰 활용)
            - 배너 메시지: body 또는 subtitle 관련 토큰 사용
            - 비슷한 크기의 토큰 선택하여 사용(하드코딩 금지)

==============================================

핵심요구사항) 배너 UI 디자인을 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.

        1) UI 구조 요구사항
            - 1. wrapper 영역
                - 화면 상단에 고정되는 배너
                - 전체 너비 (width: 100%)
                - 고정 높이 (height: 50px ~ 60px)
                - 배경색: primary 또는 brand 색상 사용
                - z-index를 설정하여 다른 요소 위에 표시
                
            - 2. content 영역
                - 최대 너비 제한 (max-width: 1200px 또는 비슷한 값)
                - 중앙 정렬 (margin: 0 auto)
                - 좌우 패딩 (padding: 0 20px)
                - flexbox 사용하여 좌우 정렬 (justify-content: space-between)
                - 세로 중앙 정렬 (align-items: center)
                
            - 3. message 영역
                - 텍스트색: white 또는 대비되는 색상
                - 폰트 크기: body 또는 subtitle 토큰 사용
                - 폰트 웨이트: 500 또는 600 (medium ~ semibold)
                - 예시 메시지: "🎉 특별 이벤트! 첫 숙박권 구매 시 10% 할인 혜택"
                
            - 4. closeButton 영역
                - 투명한 배경의 버튼
                - 닫기 아이콘(X) 또는 텍스트 표시
                - 호버 시 배경색 또는 투명도 변경
                - 커서: pointer
                - 패딩: 8px

        2) Next.js Image 컴포넌트 사용
            - close.png 아이콘을 Next.js Image 컴포넌트로 import
            - width, height 속성 지정
            - alt 속성 필수 ("닫기")

        3) 스타일링 세부사항
            - 모든 색상은 global.css의 CSS 변수 토큰 사용
            - 하드코딩된 색상 코드(#FFFFFF 등) 사용 금지
            - transition 효과로 부드러운 인터랙션
            - box-shadow로 입체감 추가 (선택사항)

        4) 반응형 고려사항
            - PC 화면 기준 구현
            - 모바일은 추후 대응 (현재 단계에서는 제외)

        5) 예시 메시지 내용
            - "🎉 특별 이벤트! 첫 숙박권 구매 시 10% 할인 혜택"
            - 또는 "✈️ 여행 이야기를 공유하고 포인트를 받으세요!"
            - 이모지를 활용하여 시각적 효과 추가
